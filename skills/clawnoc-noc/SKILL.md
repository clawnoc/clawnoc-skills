# NOC 全能包

整合 ClawNOC 全部运维技能，适合中小团队快速上手。

## 包含技能

| 技能 | 能力 |
|------|------|
| 🌐 CDN 管理 | 缓存清理、命中率分析、回源链路追踪 |
| 🔒 安全扫描 | API Key 审计、供应链检查、安全组扫描、证书监控 |
| 📊 监控告警 | 系统资源监控、告警配置、异常自动处理、多渠道通知 |
| 🚀 部署检查 | 部署验证、健康检查、灰度发布、自动回滚 |
| 📋 文档生成 | 变更记录、架构图、复盘报告、知识库更新 |

## 快速开始

```bash
# 安装全能包
claw skill install clawnoc/clawnoc-noc
```

## 使用场景

### 日常巡检
```
"检查所有服务器的 CPU、内存、磁盘状态"
"扫描所有 SSL 证书的到期时间"
"检查安全组是否有异常开放的端口"
```

### 故障处理
```
"网站返回 502，帮我排查"
"数据库连接数异常，帮我分析"
"CDN 缓存没更新，帮我清理并验证"
```

### 部署运维
```
"帮我检查部署环境是否就绪"
"部署完成后做一次全面健康检查"
"生成今天的变更记录文档"
```

### 安全审计
```
"扫描所有项目的 API Key 权限"
"检查 GitHub Actions 是否有供应链风险"
"审计 SSH 登录记录，有没有异常"
```

## 工作流程

```
用户描述问题
    ↓
Agent 识别问题类型
    ↓
调用对应技能排查
    ↓
执行修复操作
    ↓
验证修复结果
    ↓
生成文档记录
    ↓
更新知识库
```
